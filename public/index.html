Inbox

This is a small demonstrator server for an LDN Inbox. The inbox can be found here.

A notification message can be send to ths inbox by using the following curl command: 

curl -X POST -H 'Content-Type: application/ld+json' \
    --data-binary 'path_to_notification.jsonld' http://localhost:5051/inbox/
